Основное:
Приложение расположено по ссылке - https://main.d1gowrt8n348ip.amplifyapp.com/ 
Сборка и деплой производиться автоматически после пуша на гитхаб.
Серверная часть не задеплоена. Для анализа работы серверной части, необходимо развернуть приложение локально. Инструкция ниже.

Точки входа(для разработчика): 
клиентская часть - путь к файлу - dragons/public/index.html 
серверная часть -  путь к файлу - dragons/server/index.js

Как запускать приложение локально:
загрузить все файлы с репозитория https://github.com/IgorKrot/dragons на локальный компьютер в папку dragons
установить, создать и запустить базу данных PostgreSQL
данные необходимые для коннекта приложения к базе размещены в файле /dragons/server/.env . Для того, чтобы приложение общалось с базой, необходимо заполнить значения в полях, заменив их на свои:
DB_NAME={dragons} //имя БД
DB_USER={postgres} //имя пользователя БД
DB_PASSWORD={sddfsdfsdf444} //пароль доступа к БД
открыть консоль, перейти в директорию /dragons и запустить команду npm install
открыть консоль, перейти в директорию /dragons/server и запустить команду npm install
открыть консоль, перейти в директорию /dragons и запустить команду npm start
открыть консоль, перейти в директорию /dragons/server и запустить команду npm run dev
